# Car Wash IoT Dashboard

A premium dark-mode IoT monitoring dashboard for coin-operated car wash franchises.

## Tech Stack

- React 19
- TypeScript
- Tailwind CSS
- Recharts
- Vite

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the app:
   ```bash
   npm run dev
   ```
